
# Prototype Build PHP Apache Module


## How to build

### prepare

``` sh
$ ./prepare.sh
```


### download

``` sh
$ ./download.sh
```


### extract

``` sh
$ ./extract.sh
```


### configure

``` sh
$ ./configure.sh
```


### build

``` sh
$ ./build.sh
```


### install

``` sh
$ ./install.sh
```


### mod-install

``` sh
$ ./mod-install.sh
```


### mod-enable

``` sh
$ ./mod-enable.sh
```


### ini-development

``` sh
$ ./ini-development.sh
```


### apache-restart

``` sh
$ ./apache-restart.sh
```


### info-install

$ ./info-install.sh
